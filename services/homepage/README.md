# Homepage Dashboard Service Module

This module deploys [Homepage](https://gethomepage.dev) behind Traefik using a
generated Docker Compose file and declarative YAML config files.

## Deployment model

- Runtime compose file is generated by the Nix module and written to:
  - `/etc/homepage/docker-compose.yml`
- Homepage config files are generated declaratively under:
  - `/etc/homepage/config/`
- systemd runs `docker compose up -d` / `docker compose down` from
  `/etc/homepage`.
- Optional Docker socket integration can be enabled to let Homepage show local
  container-backed status data for cards.

## Config files

- `settings.yaml` is generated from `services.homepageDashboard.config.settings`
  and merged with a small default title/description.
- `services.yaml`, `bookmarks.yaml`, and `widgets.yaml` are generated directly
  from the matching module options.
- `docker.yaml` is generated from `services.homepageDashboard.config.docker` and
  is augmented automatically when Docker integration is enabled.

## Known host-specific override

- `nix-pi/nixos/hosts/private/rpi-box-02.nix` intentionally overrides
  `/etc/homepage/config/docker.yaml` with a multi-host Docker inventory so the
  Homepage instance on `rpi-box-02` can query remote Docker APIs on other
  homelab nodes.
- During incident work, do not assume the generated shared-module shape is the
  exact runtime shape on `rpi-box-02`.
- Canonical host-side reference:
  - `../nix-pi/docs/HOST_RUNTIME_DIVERGENCES.md`

## Image pinning strategy

- Default policy is pinned tags only.
- Default image is `ghcr.io/gethomepage/homepage:v1.8.0`.
- Mutable tags like `latest` are blocked unless
  `services.homepageDashboard.image.allowMutableTag = true`.

## Example

```nix
services.homepageDashboard = {
  enable = true;
  hostname = "homepage.${config.lab.domain}";
  tls = true;

  docker.enable = true;

  config = {
    settings = {
      title = "Homelab";
      description = "Internal dashboard";
    };

    services = [
      {
        "Core" = [
          {
            "Grafana" = {
              href = "https://grafana.${config.lab.domain}/";
              description = "Metrics and dashboards";
              server = "local";
              container = "grafana";
            };
          }
        ];
      }
    ];
  };
};
```
